# Projecto Final - DEFESA 1

Crie uma nova função que altera o estado de todas as bombas para o estado off. Essa função deve ser acessível através do menu com a opção "hell":

```
+-----------------------------------------------------
read <filename>     - read input file
show                - show the mine map
trigger <x> <y>     - trigger mine at <x> <y>
log <x> <y>			- trigger mine at <x> <y>
plant <x> <y>       - place armed mine at <x> <y>
export <filename>   - save file with current map
quit                - exit program
sos                 - show menu
hell				- explode all existing bombs
+-----------------------------------------------------
```

Esta opção não faz trigger nem log, apenas altera o estado das bombas. Antes de regressar ao menu, o mapa deve ser apresentado no terminal.



Submeta o seu código alterado no PANDORA no contest LP12020PFD1.
